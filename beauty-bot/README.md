# Kira — твой AI Бьюти-стилист 💅✨

Привет! Это проект Telegram-бота, который поможет тебе с выбором стиля, уходом за кожей и анализом внешности с помощью искусственного интеллекта (Google Gemini).

## 🚀 Как запустить бота у себя на компьютере

### 1. Подготовка
Тебе понадобится установленный [Node.js](https://nodejs.org/).

### 2. Установка зависимостей
Открой терминал в папке проекта и выполни команду:
```bash
npm install
```
Это установит все необходимые библиотеки.

### 3. Настройка ключей
Тебе нужно создать файл `.env` (или переименовать `.env.example`) и вписать туда свои ключи:

1.  **Telegram Bot Token:**
    *   Напиши [BotFather](https://t.me/BotFather) в Telegram.
    *   Создай нового бота (`/newbot`) и следуй инструкциям.
    *   Скопируй полученный API Token и вставь его в поле `BOT_TOKEN`.
2.  **Gemini API Key:**
    *   Зайди в [Google AI Studio](https://aistudio.google.com/).
    *   Создай бесплатный API ключ.
    *   Вставь его в поле `GEMINI_API_KEY`.

### 4. Запуск локально
В терминале выполни:
```bash
npm start
```
Если всё сделано правильно, ты увидишь сообщение: `✅ Бот Kira запущен!`.

## ☁️ Как задеплоить на Railway (в интернет)

1.  Загрузи свой код на **GitHub** (убедись, что файл `.env` **не** попал в репозиторий, он должен быть в `.gitignore`).
2.  Зайди на [railway.app](https://railway.app/) и создай новый проект.
3.  Выбери **"Deploy from GitHub repo"** и укажи свой репозиторий.
4.  После создания проекта зайди во вкладку **Variables** (Переменные).
5.  Добавь две переменные:
    *   `BOT_TOKEN` — твой токен от BotFather.
    *   `GEMINI_API_KEY` — твой ключ от Google.
6.  Railway сам запустит бота, и он будет работать 24/7!
