# Kira Beauty Bot ✨

Telegram AI бьюти-стилист на базе Node.js, grammY и Google Gemini 2.5 Flash.

Кира — твоя честная подруга-стилист, которая без лишних восторгов разберет твой образ, подскажет подходящие цвета и честно скажет, если вещь тебя не красит.

## Возможности

- ✨ **Анализ фото**: Силуэт, цветотип и конкретные рекомендации.
- 👗 **Собрать образ**: Идеи для твоего настроения.
- 🎨 **Цвета**: Твоя палитра и почему она работает.
- 🪞 **Что мне идет**: Конкретика по фасонам, тканям и украшениям.
- 🚫 **Что не носить**: Честный разбор вещей, которые лучше оставить в магазине.
- ☕ **Честный разбор**: Без лести о твоем стиле.
- 🛍️ **Капсула**: Твой базовый мини-гардероб.

## Как настроить

### 1. Получение Telegram токена
1. Найдите [@BotFather](https://t.me/botfather) в Telegram.
2. Отправьте команду `/newbot` и следуйте инструкциям.

### 2. Получение Gemini API Key
1. Перейдите в [Google AI Studio](https://aistudio.google.com/).
2. Получите API key для модели `gemini-2.5-flash`.

## Локальный запуск

1. Клонируйте репозиторий.
2. Создайте `.env` (см. `.env.example`).
3. `npm install`
4. `npm start`
