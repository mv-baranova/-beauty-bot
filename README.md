# Kira Beauty Bot ✨

Telegram AI бьюти-стилист на базе Node.js, grammY и Google Gemini 2.5 Flash.

Кира — это современный AI-ассистент, созданный для того, чтобы помогать тебе выглядеть и чувствовать себя великолепно. Она использует передовые технологии для анализа стиля, подбора макияжа и создания уникальных образов в стиле luxury / Pinterest.

## Возможности

- ✨ **Анализ фото**: Загрузи свое фото, и Кира даст подробные советы по макияжу, прическе и стилю.
- 👗 **Подобрать образ**: Получи рекомендации по одежде и аксессуарам.
- 💄 **Советы по макияжу**: Узнай, какие бьюти-тренды подходят именно тебе.
- 🎨 **Цвета и стиль**: Подбор идеальной цветовой палитры.
- 💬 **AI Чат**: Общайся с Кирой на любые темы, связанные с красотой.

## Как настроить

### 1. Получение Telegram токена
1. Найдите [@BotFather](https://t.me/botfather) в Telegram.
2. Отправьте команду `/newbot` и следуйте инструкциям.
3. Сохраните полученный API Token.

### 2. Получение Gemini API Key
1. Перейдите в [Google AI Studio](https://aistudio.google.com/).
2. Нажмите "Get API key".
3. Создайте новый API key.

## Локальный запуск

1. Клонируйте репозиторий.
2. Создайте файл `.env` в корневой папке и добавьте ваши ключи (см. `.env.example`):
   ```env
   BOT_TOKEN=ваш_токен_телеграм
   GEMINI_API_KEY=ваш_ключ_gemini
   ```
3. Установите зависимости:
   ```bash
   npm install
   ```
4. Запустите бота:
   ```bash
   npm start
   ```

## Деплой (Railway / Render)

1. Подключите ваш GitHub репозиторий к платформе.
2. В разделе **Environment Variables** добавьте:
   - `BOT_TOKEN`
   - `GEMINI_API_KEY`
3. Команда запуска: `npm start` (или `node bot.js`).
4. Бот автоматически запустится и начнет работу через polling.

## Troubleshooting

- **Бот не отвечает**: Проверьте логи на Railway/Render. Убедитесь, что `BOT_TOKEN` указан верно.
- **Ошибка Gemini API**: Убедитесь, что ваш `GEMINI_API_KEY` активен и у вас есть доступ к модели `gemini-2.5-flash`.
- **Проблемы с фото**: Бот использует `getFile` API Telegram. Убедитесь, что у бота достаточно прав.

## Структура проекта

- `handlers/` — обработка команд, текста и фото.
- `services/` — интеграция с Gemini API.
- `config/` — конфигурация и валидация окружения.
- `prompts/` — системные промпты для AI.
- `utils/` — вспомогательные функции (загрузка файлов).
- `bot.js` — точка входа.
